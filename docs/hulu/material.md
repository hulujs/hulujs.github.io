# hulu material <Badge>待建设</Badge>

## 简介

管理物料的相关命令

## 使用

```bash
hulu material <command>
```

## 命令

### repo

创建一个物料repo, 编写物料

```bash
hulu material repo <name>
```

### push

添加物料到物料市场

```bash
hulu material push <name>
```

### remove

移除物料

```bash
hulu material remove <name>
```

### download

从物料市场下载物料到本地

```bash
hulu material download <name>
```
